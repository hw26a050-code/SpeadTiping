# 「スピード耐久タイピング」ゲームシステム・ルール仕様書

## 1. 概要
本ゲームは、高速タイピングを練習するための、時間制限付きのエンドレス・タイピングゲームです。日本語の基本単語をローマ字で入力し、タイピングミスを避けてハイスコアを目指します。

## 2. 基本ルール
- **入力方式**: 日本語ローマ字入力（複数パターン入力対応）。
- **使用単語**: 日常的な基本単語のみ。
- **ゲーム形式**: エンドレス形式。ミスによるゲームオーバー、または制限時間切れになるまで単語が次々と出題されます。

## 3. スコアシステム
- タイピングが完了した単語（正解数）をスコア（1単語 = 1点）とします。

## 4. ライフシステム
- **初期ライフ**: 3（許容タイプミス数）
- **減少条件**: 1文字のタイプミスごとにライフが1減少します。**また、ミスをした時点でその単語の入力は強制中断され、即座に次のお題（単語）へ移行します。**
- **ゲームオーバー条件**: ライフが0になると、その場で即ゲームオーバーとなります。

## 5. 制限時間システム
- 各単語ごとに個別に制限時間が設定されます。
- **制限時間の計算式**: 
  - **Easy**: `制限時間(秒) = 出題単語のひらがな文字数 * 2.5` （小数第二位以下繰り上げ）
  - **Normal**: `制限時間(秒) = 出題単語のひらがな文字数 * 0.8` （小数第二位以下繰り上げ）
  - **Hard**: `制限時間(秒) = 出題単語のひらがな文字数 * 0.4` （小数第二位以下繰り上げ）
- **時間切れ時のペナルティ**:
  - **Easy**: ライフが1減少し、次の単語へ移行
  - **Normal**: ライフが1減少し、次の単語へ移行
  - **Hard**: その時点で即ゲームオーバー

## 6. 難易度（難易度区分）
プレイヤーはゲーム開始前のタイトル画面にて、以下の3つの難易度から1つを選択してプレイを開始します。

| 難易度 | 条件（出題される文字数） | 時間切れ時ペナルティ | 備考 |
| :--- | :--- | :--- | :--- |
| **Easy** | ひらがな日本語4文字以下 | ライフ1減少して次へ | 初心者向け |
| **Normal** | ひらがな日本語5文字以上9文字以内 | ライフ1減少して次へ | 中級者向け |
| **Hard** | ひらがな日本語10文字以上 | 即ゲームオーバー | 上級者向け |

## 7. ローマ字複数入力サポート
一般的なローマ字の複数パターン入力をサポートします。
- 例：「し」に対する `si` / `shi`
- 例：「つ」に対する `tu` / `tsu`
- 例：「じ」に対する `zi` / `ji`
- その他、一般的なヘボン式・訓令式などのバリエーションに対応。
- **「ん」の入力について**: 画面に表示されるガイド（`nn`）と実際の入力体験を完璧に一致させるため、一文字の `n` による簡易入力は認めず、常に `nn` （nキーを2回打つ方式）に固定して厳密な判定を行っています。

## 8. キーボード入力制御（物理キー押し下げロック機構）
キーの長押しやチャタリング、同時押しによる意図しない重複入力を防ぐため、以下の制御を行います。
- **物理キーベースのキーアップ検出（e.code Release Lock）**: 
  - `keydown` イベントを検知した時点で、その物理キー（`e.code`。Shiftキーなどの影響を受けない物理的な位置情報）を「押し下げ中（Pressed）」のセットに登録し、判定処理を1回のみ実行します。
  - その物理キーが離されて `keyup` イベントが発火するまで、同一キーの新たな `keydown` イベントを完全に無視（ロック）します。
  - 修飾キーの影響による大文字小文字のズレや、チャタリング、長押しによるブラウザのリピートイベントを完全に遮断します。
  - ウィンドウのフォーカスアウト（`blur`）時や、単語クリア・シーン切り替え時にも、この押し下げ状態（ロック）を自動的に一括リセットします。
- **目的**: これにより、「飛行船（hikousenn）」の `nn` など同じ文字が連続する単語であっても、1回のタイプ（キー長押しや連続的な物理チャタリング）で同時に2文字分が入力されて判定が狂ってしまうバグを防ぎ、1回ずつキーを離して確実にタイピングする正確で公平なゲーム体験を保証します。
